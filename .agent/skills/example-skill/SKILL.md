# Example Skill

Esta es una plantilla para crear skills. Según la documentación de AntiGravity, para que la skill funcione:
1. Debe estar en un directorio como `.agent/skills/<nombre-skill>/` (local) o `~/.gemini/antigravity/skills/<nombre-skill>/` (global).
2. Debe contener este archivo `SKILL.md` con un encabezado de metadatos YAML (frontmatter) que incluya al menos el campo `description` en tercera persona.
3. El cuerpo del archivo puede incluir instrucciones paso a paso, escenarios de uso y guía para el agente.

Para usar esta plantilla, modifica este archivo añadiendo las instrucciones detalladas de tu nueva agentic skill.
